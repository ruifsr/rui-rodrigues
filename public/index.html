About Me

Rui Filipe dos Santos Rodrigues

Country/Region: Portugal, Setúbal
Ocupations:
  • Software Developer
  • Assistant Professor
  • PhD Student
I started my professional career by working at Rieter Automotive (today known as Autoneum), where I was in charge of systems maintenance, as well as managing the network system and creating information backups.
After I started working in INSTICC where I've developed several features for PRIMORIS (Event Management System). I was also responsible/developer for several web crawling projects. I've also developed other event management systems (for APMGF (Medical), ISAF Youth (Sport) and other fields of activity).
Since 2016 I’m an assistant professor at Polytechnic Institute of Setúbal where I teach web programming, mobile programming and web services (.net) related courses.
Recently I enrolled on a PHD program in Computer Science at Nova University of Lisbon. Meanwhile I'm also working in "CultureMoves" european project, under which we've been developing a web-annotator.
